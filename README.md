# oplist
A simple pocketmine oplist plugin

Commands:
/oplist


You need to either be OP or be on console/rcon or have the perm "oplist.command" to run the command.


The response you get when you use the command will look like this:

\-------------------------

Opped User List:

xdolmation1

xdolmation2

xdolmation3

\-------------------------
